Sistemas Operacionais

Windows 1.0 (1985)

A primeira versão do SO foi lançada no longínquo ano de 1985 e era mais uma extensão do MS-DOS do que um software completo. Ele se caracterizava pelo modelo colorido na tela e aplicações como bloco de notas e até um Paint

Windows 2.0 (1987)

Windows 2.0 foi uma interface gráfica de usuário entre o sistema operacional Microsoft DOS e o computador, é a segunda versão da família Windows desenvolvido pela empresa norte-americana Microsoft lançado em 9 de dezembro de 1987, sucessor da versao Windows 1.x.

Windows 3.0 (1990)

O Windows 3.0 é o terceiro grande lançamento do Microsoft Windows, lançado em 1990. Sua nova interface gráfica do usuário representa os aplicativos como ícones clicáveis, em vez da lista de nomes de arquivos de seus antecessores.

Windows 3.1 (1992)

O Windows 3.0 , o antecessor do 3.1, foi lançado em 1990 e é considerado a primeira versão do Windows a receber aclamação da crítica. O Windows 3.0 recebeu cerca de 10 milhões de vendas antes do lançamento do Windows 3.1 em 6 de abril de 1992. A Microsoft iniciou uma campanha publicitária na televisão pela primeira vez em 1º de março de 1992. Os anúncios, desenvolvidos pela Ogilvy & Mather , foram projetados para apresentar o Windows a um público mais amplo

Windows 95 (1995)

O Microsoft Windows 95 foi um sistema operacional de 16/32 bits criado pela empresa Microsoft. Lançado em 24 de agosto de 1995, o Windows 95 revolucionou o mercado de sistemas operacionais e passou a vir instalado por padrão com o MS-DOS 7.0, sendo o principal lançamento da empresa na década de 1990.

Windows 98 (1998)

O Windows 98 foi um sistema operacional com interface gráfica desenvolvido pela Microsoft. É o segundo maior lançamento da família de sistemas operacionais Windows 9x, e é o sucessor do Windows 95. Ele foi lançado no formato RTM em 15 de maio de 1998, e lançado em sua versão varejo em 25 de junho de 1998.

Windows 2000 (2000)

Microsoft Windows 2000 foi um sistema operacional preemptivo, gráfico e de fácil aprendizagem que foi desenvolvido para trabalhar com um ou mais processadores de 32 bits em computadores na arquitetura x86 e 64 bits na arquitetura IA6

Windows ME (Millennium Edition) (2000)

O Windows Millennium Edition, ou Windows ME, foi um sistema operacional de interface gráfica desenvolvido pela Microsoft, lançado em 14 de Setembro de 2000. Foi o último sistema operacional da família Windows 9x, que rodam em cima do MS-DOS

Windows XP (2001)

O Windows XP foi uma versão do Microsoft Windows, uma série de sistemas operacionais desenvolvidos pela Microsoft para computadores pessoais, incluindo computadores residenciais e de escritório, notebooks, tablets e media centers. O nome "XP" deriva de eXPerience

Windows Vista (2007)

O Windows Vista foi um sistema operacional desenvolvido pela Microsoft, sendo a sexta versão do Windows para uso em computadores pessoais, incluindo computadores residenciais e de escritórios, laptops, Tablet PCs e computadores Media Centers. Antes do seu anúncio em 22 de Julho de 2005, o Windows Vista era conhecido pelo nome de código Longhorn.

Windows 7 (2009)

Windows 7 foi uma versão do Microsoft Windows, uma série de sistemas operativos produzidos pela Microsoft para uso em computadores pessoais, incluindo computadores domésticos e empresariais, laptops, tablets e PCs de centros de mídia, entre outro.

Windows 8 (2012)

O Windows 8 foi uma versão do Microsoft Windows, uma série de sistemas operacionais desenvolvidos pela Microsoft para computadores pessoais, laptops e tablets. É o sucessor do Windows 7

Windows 8.1 (2013)

Windows 8.1 (codinome Windows Blue[2]) foi uma versão do Microsoft Windows, série de sistemas operacionais comercializados pela Microsoft para computadores, tablets e laptops. Foi o sucessor do Windows 8. Foi anunciado no dia 14 de maio de 2013 e a versão final foi lançada e disponibilizada para consumidores e para o público em geral em 17 de outubro de 2013.

Windows 10 (2015)

Windows 10 é uma versão do sistema operacional da Microsoft disponível para baixar em PC e em notebook.

Windows 11 (2021)

O Windows 11 é um sistema operacional e a versão atual do sistema Microsoft Windows (série de sistemas operacionais para computadores pessoais, laptops e tablets) desenvolvido pela Microsoft, anunciada e lançado em 2021, como o sucessor da versão Windows 10 (lançado seis anos antes).

1.Distribuição Populares

O Linux Mint é uma distribuição baseada em Ubuntu que vem se popularizando há tempos. Sendo bastante usada por quem busca um sistema com bom desempenho mesmo em máquinas mais modestas, ele conta com os principais recursos esperados do sistema operacional e é capaz de suprir praticamente qualquer demanda do usuário.

Linux Mint

Linux Mint. Linux Mint é uma popular distribuição Linux conhecida por sua interface amigável e facilidade de uso, especialmente entre os novos usuários de Linux.

Debian

O Debian é um sistema operacional Linux utilizado em laptops, desktops e servidores. Ele é, provavelmente, a maior distribuição Linux não comercial, tanto em volume de desenvolvedores quanto em número de usuários, diretos e indiretos.

Fedora

Fedora (conhecido como Fedora Core antes da versão 7) é um sistema operacional ou sistema operativo Linux. O sistema operacional Fedora Linux é software livre e de código aberto, e os programas disponíveis dentro de seu repositório de programas também são programas livres que aderem a uma licença livre.

OpenSUSE

O openSUSE cria uma das melhores distribuições Linux do mundo, além de uma variedade de ferramentas como OBS, OpenQA, Kiwi, YaST, OSEM, trabalhando junto em um estilo aberto, transparente e amigável, como parte de uma comunidade mundial de Software Livre e de Código Aberto.

2.Distribuições para Avançadas

O Arch Linux é uma distribuição GNU/Linux x86_64 de uso geral desenvolvida independentemente que se empenha em fornecer as últimas versões estáveis da maioria dos softwares seguindo um modelo de lançamento contínuo (rolling-release

Gentoo

Gentoo Linux é conhecido por sua flexibilidade, permitindo aos usuários personalizarem e otimizarem seu sistema desde a compilação do código-fonte. Essa abordagem torna o Gentoo uma escolha poderosa para usuários avançados que desejam controle total sobre o funcionamento do sistema.

Slackware

O Slackware Linux, criado por Patrick Volkerding em 1992, é a distribuição mais antiga ainda existente. Desenvolvido a partir do descontinuado "Projeto SLS", o Slackware 1.0 era distribuído em 24 disquetes e construído em cima da versão 0.99pl11-alpha do kernel Linux.

3.Distribuições para Servidores

CentOS Stream é uma distribuição Linux® em que membros da comunidade open source podem desenvolver, testar e contribuir para um upstream de distribuição de entrega contínua para o Red Hat® Enterprise Linux, tudo isso junto aos desenvolvedores da Red Hat.

Debian

O Debian é uma distribuição Linux conhecida por sua estabilidade, segurança e ampla seleção de pacotes. É frequentemente utilizado como base para outras distribuições, incluindo o Ubuntu. O Debian é recomendado para servidores que exigem confiabilidade e um sistema operacional flexível e personalizável.

AlmaLinux e Rocky Linux

O AlmaLinux segue na lista como o preferido, o projeto surgiu quando a Red Hat resolveu acabar com o CentOS. O projeto foca em ser totalmente compatível com os softwares do Red Hat Enterprise Linux. Rocky Linux é uma distribuição Linux recentemente lançada, que visa fornecer uma alternativa de código aberto ao CentOS, após suas mudanças de direção.

4.Distribuições Leves

O Lubuntu é uma das distros leves mais conhecidas e é baseada no Ubuntu, o que significa que herda sua enorme comunidade e suporte. Lubuntu utiliza o LXQt, uma interface gráfica extremamente leve e responsiva. Mesmo em hardware mais antigo, você pode esperar um bom desempenho.

Xubuntu

O Xubuntu, como o nome sugere, usa o XFCE, uma interface leve e altamente personalizável. Apesar de ser leve, o XFCE oferece uma experiência visual agradável e moderna. Xubuntu é estável e eficiente, tornando-o adequado tanto para computadores de baixo desempenho quanto para usuários que preferem um sistema menos pesado, mas ainda assim poderoso.

Puppy Linux

O Puppy Linux é uma distribuição incrivelmente leve, projetada para rodar em qualquer tipo de hardware. Ela pode ser executada diretamente de um pendrive ou CD, sem necessidade de instalação.

Bodhi Linux

O Bodhi Linux é único em sua abordagem, utilizando a interface Moksha, baseada no Enlightenment. Ela é extremamente leve e visualmente atraente. Bodhi Linux requer apenas 512MB de RAM, tornando-o uma excelente escolha para quem tem computadores antigos, mas quer uma interface mais sofisticada.

5.Distribuições para Privacidade e Segurança

Tails (The Amnesic Incognito Live System) é uma distribuição Linux voltada pra Live CD's baseada em Debian que veio para garantir a segurança e o anonimato de seus utilizadores enquanto navegam pela internet, impedindo que o mesmo deixe rastros em seus acessos utilizando o tráfego da Rede Tor e criptografia PGP.

Qubes OS

Qubes OS é um sistema operacional gratuito e de código aberto criado para aumentar a segurança em computação, especialmente em desktop de um único usuário.

Kali Linux

Kali Linux é uma distribuição Linux baseada em Debian, projetada especificamente para testes de penetração e auditoria de segurança.

Edubuntu

Edubuntu é um esforço comunitário que busca trazer a liberdade do desktop Linux e a vasta biblioteca de software educacional de código aberto para a sala de aula. Uma distribuição destinada a usuários de 6 a 18 anos e projetada para fácil instalação e manutenção contínua do sistema.

Debian Edu/Skolelinux

O Debian Edu, também conhecido como Skolelinux, é uma distribuição Linux baseada no Debian que fornece um ambiente de rede escolar pronto e completamente configurado. Imediatamente após a instalação, um servidor escolar executando todos os serviços necessários em uma rede escolar é configurado, aguardando somente a adição de usuários(as) e máquinas via GOsa², uma confortável interface web.

Scientific Linux

Scientific Linux (versão atual: SL 7.9, publicado em 20.10.2020) é uma distribuição Linux padronizado pelo Fermi National Accelerator Laboratory ou Fermilab (Illinois/USA). Ele é um sistema operacional livre e de código aberto baseado no Red Hat Enterprise Linux (RHEL) e pretende ser o mais próximo possível da distribuição de empreendimento comercial.

7.Distribuições para Desenvolvimento

Manjaro é uma distribuição Linux amigável baseada no sistema operacional Arch desenvolvido de forma independente. Dentro da comunidade Linux, o próprio Arch é conhecido por ser uma distribuição excepcionalmente rápida, poderosa e leve que fornece acesso aos softwares mais recentes e de última geração

Pop!_OS

Pop!_ OS é uma distribuição Linux gratuita de código aberto, baseada no Ubuntu, e conta com um desktop GNOME personalizado. A distribuição é desenvolvida pela fabricante estadunidense de computadores Linux, chamada System76.

1.Mac OS Clássico(1984-2001)

O macOS é a segunda grande série de sistemas operacionais Macintosh. O primeiro é coloquialmente chamado de Mac OS Classic, introduzido em 1984, o último lançamento desse sistema foi o Mac OS 9 em 1999. A primeira versão para desktop, o Mac OS X 10.0, foi lançada em março de 2001, com sua primeira atualização, a 10.1, chegando ainda naquele ano.

System 6 (1988)

Sistema 6 ainda era amplamente considerada a melhor versão de software de sistema para o Macintosh. A multitarefa cooperativa fez sua estreia no Macintosh em 1985 com um programa chamado switcher, que permitia ao usuário iniciar vários aplicativos e alternar entre eles.

System 7 (1991)

Apresentava multitarefa cooperativa integrada, memória virtual, compartilhamento de arquivos pessoais, uma interface com aparência ligeiramente 3D aproveitando as cores, quicktime para captura e reprodução de vídeo e quickdraw 3D para gráficos 3D.

Mac OS 8 (1997)

Mac OS 8 a primeira versão ainda suportava macs de "primeira geração" com processadores motorola 680x0-family; suas versões posteriores rodavam em sistemas powerpc G3, como o powerbook G3 wallstreet.

Mac OS 9 (1999)

Mac OS 9 foi a última versão do sistema operacional macintosh "clássico" (Mac OS) lançada antes de ser sucedido pelo Mac OS X. O Mac OS 9 foi considerado a versão mais funcional do Mac OS original.

2. Mac OS X(2001-2012)

Mac OS X 10.1 Puma (2001)

O Mac OS X 10.1 "Puma" foi a segunda versão do sistema operacional Mac OS X, para computadores desktop e servidor da Apple Inc. Ele substituiu o Mac OS X 10.0 e antecedeu o Mac OS X 10.2 "Panther". A versão 10.1 foi lançada em 25 de setembro, 2001 como uma macro atualização gratuita da versão 10.0. A partir da versão 10.1.2, a Apple Inc. fez do Mac OS X o sistema operacional padrão dos novos Macs.

Mac OS X 10.2 Jaguar (2002)

Mac OS X Jaguar (versão 10.2) foi a segunda versão do Sistema Operacional para desktop e servidores da série Mac OS X. Substituiu o Mac OS X v10.1 "Puma" e antecedeu o Mac OS X v10.3 “Panther”.

Mac OS X 10.3 Panther (2003)

O Mac OS X 10.3 "Panther" é a quarta versão do sistema operacional Mac OS X da Apple Inc. para computadores desktop e servidor. Substituiu o Mac OS X 10.2 "Jaguar" e antecedeu o Mac OS X 10.4 "Tiger". A Apple Inc. libertou o Panther em 24 de outubro, 2003. O custo era de US$129 para uma licença de um único-usuário e US$ 199 para uma licença do tipo "Pacote Família " (5 computadores, 1 residência).

Mac OS X 10.4 Tiger (2005)

O Mac OS X 10.4 "Tiger", foi o quinto lançamento do Mac OS X. O Tiger foi liberado ao público em 29 de abril de 2005 como o sucessor do Mac OS X 10.3 "Panther", que foi lançado 18 meses antes, sendo substituido pelo Mac OS X 10.5 "Leopard", em 26 de outubro de 2007, após 30 meses, tornando-se o Mac OS X de mais longa versão.

Mac OS X 10.5 Leopard (2007)

O Mac OS X 10.6 "Leopard" é o sexto lançamento do Mac OS X , da Apple, desktop e servidor de sistema operacional para computadores Macintosh. Leopard foi lançado em 26 de outubro de 2007 como o sucessor do Tiger (versão 10.4), e está disponível em duas variantes: uma área de trabalho versão adequada para computadores pessoais , e um servidor de versão, o Mac OS X Server

Mac OS X 10.6 Snow Leopard (2009)

O Mac OS X 10.6 "Snow Leopard" é um sistema operacional da Apple sucessor da versão 10.5 "Leopard". Steve Jobs, antigo CEO da Apple, anunciou o Snow Leopard na Apple Worldwide Developers Conference (WWDC) em 8 Junho de 2008

Mac OS X 10.7 Lion (2011)

Mac OS X v10.7 "Lion" é uma das versões do sistema operacional Mac OS X da Apple Inc., sendo o sucessor do Snow Leopard e antecessor do Mountain Lion, e foi lançado mundialmente em 20 de Julho de 2011 através da Mac App Store.

Mac OS X 10.8 Mountain Lion (2012)

Apple lançou a última versão do seu sistema operacional: o Mac OS X 10.8 Mountain Lion. Esta nova plataforma inclui mais de 200 novas características completamente integradas, cuja intenção é fazer cada vez mais simples seu trabalho e sua vida. Algumas das características destacadas incluem um iCloud integrado, que mantém todo seu conteúdo atualizado e sincronizado com o resto dos seus produtos Apple; uma aplicação de ditado que faz do teclado algo opcional, e por último destacamos um centro de notificações unificado que ajuda a manter-se atualizado em relação a tudo o que acontece no seu computador.

3. MacOS (2012-Presente)

OS X v10.9 Mavericks é a décima versão do OS X, sistema operacional para Mac. OS X Mavericks foi anunciado em 10 de junho de 2013 e lançado no dia 22 de Outubro de 2013 como uma atualização gratuita através da Mac App Store.

OS X 10.10 Yosemite (2014)

osemite é a décima primeira versão do OS X, sistema operacional para Mac. OS X Yosemite foi anunciado e liberado para desenvolvedores em 2 de junho de 2014, na WWDC. O sistema foi liberado de graça para todos os usuarios de Mac no dia 16 de outubro de 2014, por download na App Store.

OS X 10.11 El Capitan (2015)

OS X v10.11 El Capitan é a décima segunda versão do OS X, sistema operacional para Mac. O sistema operacional foi liberado de graça para todos os usuários de Macs no dia 30 de setembro de 2015, por download na App Store.

macOS 10.12 Sierra (2016)

macOS Sierra é a décima terceira versão do sistema operacional macOS (anteriormente OS X) da Apple, que a partir desta versão passou-se a chamar macOS com objetivo de padronizar os nomes dos sistemas operacionais da empresa.

macOS 10.13 High Sierra (2016)

macOS 10.13 High Sierra é uma versão do sistema operacional para computadores da Apple, lançada em 25 de setembro de 2017. Focada em aprimoramentos de desempenho e segurança, o High Sierra introduziu o sistema de arquivos Apple File System (APFS) para melhor eficiência e segurança de dados.

macOS 10.14 Mojave (2018)

macOS 10.14 Mojave é uma versão do sistema operacional da Apple para Macs, lançada em setembro de 2018. Destaca-se pelo novo Modo Escuro, que oferece uma interface visual mais confortável para ambientes com pouca luz.

macOS 10.15 Catalina (2019)

macOS 10.15 Catalina é uma versão do sistema operacional da Apple para computadores Mac, lançada em outubro de 2019. Esta atualização trouxe diversas melhorias e novos recursos, marcando a transição para uma era mais moderna do software para Mac.

macOS 11 Big Sur(2020)

O macOS Big Sur é a nova versão do sistema operacional da Apple para computadores. A plataforma foi anunciada durante a WWDC 2020 e deve chegar para MacBooks e iMacs a partir do segundo semestre de 2020.

macOS 12 Monterey (2021)

macOS 12 Monterey é a décima segunda versão do sistema operacional para computadores Mac, desenvolvida pela Apple. Lançada oficialmente em outubro de 2021, Monterey introduz uma série de novos recursos e melhorias em relação às versões anteriores.

macOS 13 Ventura (2022)

macOS 13 Ventura é uma versão do sistema operacional da Apple para computadores Mac, lançada em outubro de 2022. Ventura introduz várias melhorias e novos recursos, como o Stage Manager, que facilita a organização de janelas e aplicativos, e a continuidade da câmera, permitindo usar o iPhone como webcam.

macOS 14 Sonoma (2023)

o macOS 14 Sonoma traz um novo posicionamento para os widgets. No macOS 13 Ventura, eles ficavam em uma barra lateral, que aparecia e desaparecia. Agora, eles podem ficar na área de trabalho. Assim, é mais fácil de acompanhar informações, como previsão do tempo, lembretes, horários e muito mais.
