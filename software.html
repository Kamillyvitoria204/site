Conceito de Software

1.SISTEMAS OPERACIONAIS

Linux é um Sistema Operacional, assim como o Windows e o Mac OS, que possibilita a execução de programas em um computador e outros dispositivos

Distribuições

Uma distribuição Linux é um sistema operativo ou sistema operacional criado a partir de uma coleção de softwares, com o uso do núcleo Linux, um sistema gestor de pacotes, e um repositório de programas

Apple

O Apple File System (APFS) é um sistema de arquivos proprietário que foi projetado levando em consideração a criptografia. O APFS funciona em todas as plataformas da Apple: no iPhone, iPad, Mac, Apple TV e Apple Watch.

Microsoft

Microsoft Corporation é uma empresa transnacional dos Estados Unidos com sede em Redmond, Washington, que desenvolve, fabrica, licencia, apoia e vende softwares de computador, produtos eletrônicos, computadores e serviços pessoais.

Software/Licenciados

O licenciamento de software funciona como um tipo de contrato estabelecido entre o fabricante (desenvolvedor) e a pessoa ou empresa que o usará. Nele, podem existir restrições para downloads, instalações, manutenções, alterações (mudanças no código-fonte), uso de certas funcionalidades e distribuição.

Livres

Nessa modalidade de licenciamento, a pessoa física ou jurídica ganha uma permissão ampla de downloads, instalações, estudo do código-fonte, modificações das funcionalidades e distribuições. Ou seja, o usuário tem total liberdade para adaptar o software às necessidades pessoais ou do negócio. Porém, isso não significa que ele será gratuito.

Proprietarios/Aplicativos de Programação

Alguns dos mais conhecidos softwares proprietários são o Microsoft Windows, o Microsoft Office, o RealPlayer, o Adobe Photoshop, o Mac OS, o WinZip, algumas versões do UNIX, entre outros.

Diagnóstico de software

O diagnóstico de software é um processo crítico que visa identificar e analisar problemas, falhas e ineficiências em sistemas de software.
